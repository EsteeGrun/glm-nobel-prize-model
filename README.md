# Nobel Prize Sharing Prediction

This project analyzes the probability that a Nobel Prize is shared using logistic regression and generalized linear models.

## Methods
- Binomial Generalized Linear Model (GLM)
- Logistic regression
- AIC model selection
- Residual diagnostics

## Tools
R  
ggplot2  
Statistical modeling

## Dataset
Nobel Prize laureate dataset (1901–2025)

## Key Findings
Prize category and number of laureates per year significantly influence the probability that a Nobel Prize is shared.
